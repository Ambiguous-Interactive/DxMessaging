Unity 2021.3+  ·  MIT  ·  zero allocation

Decoupled, simple systems.

Send a typed message and any system reacts, with no reference back to the sender. Three message types, ~10 ns per handler, zero GC.

Get started View on GitHub
$ openupm add com.wallstop-studios.dxmessaging
Player.cs 
using WallstopStudios.DxMessaging;

[DxTargetedMessage]
public partial struct Heal
{
    public int amount;
}

// register, cleanup is automatic
Token.RegisterGameObjectTargeted<Heal>(OnHeal);

// send from anywhere
new Heal(50).EmitGameObjectTargeted(player);

01 · UNTARGETED

Announce to all

A global message with no sender and no target. Anything listening hears it. The PA system.

02 · TARGETED

Command one

A message to one specific GameObject. Only that object receives it. The addressed letter.

03 · BROADCAST

Emit a fact

A fact from a known source. Listeners can filter by who sent it. The radio station.
