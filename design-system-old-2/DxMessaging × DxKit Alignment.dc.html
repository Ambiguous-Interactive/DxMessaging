Concepts ▸ Message types

Message types

DxMessaging gives you three message types. Thinking in them opens up designs that direct references never allow, and each one maps to a way people already communicate.

▸ Note
If it matters who sent it, reach for Broadcast or Targeted, never Untargeted.

Define a message

Mark a struct with an attribute. The source generator does the rest.

csharp copy
[DxBroadcastMessage]
[DxAutoConstructor]
public readonly partial struct TookDamage
{
public readonly int amount;
}
✓ Tip
Pass messages by ref in handlers to keep dispatch allocation-free on hot paths.
